git命令基本操作

git init 初始化

git status查看仓库状态

git add . 把所有文件上传暂存区

git commit -m "描述"把暂存区的修改提交到本地

git diff查看工作区修改

git diff --cached查看暂存区修改

git reset --hard commit_id(ID)返回到指定ID版本

git reset --hard HEAD返回一个版本

git reset --hard HEAD^~n返回N个版本

git clone 地址克隆

git log查看版本/关机后消失

git reglog查看版本/关机不消失

git branch -d 分支名删除本地分支

git push origin -d 分支名删除远程分支

git建立连接

git remote add origin 关联的ssh地址建立关联

git push origin master把分支 master 推送远程

git push -u origin master-u 给master分支添加默认标记 可以直接使用git push 进行推送

获取SSH秘钥

ssh-keygen
